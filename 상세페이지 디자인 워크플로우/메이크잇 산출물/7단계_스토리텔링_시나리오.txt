## 2️⃣ 스토리 전개 흐름 요약

> **공감(가격 부담) → 공감(화질 불안) → 공감(안전 이슈) → 후킹(브랜드 약속) → 해결(안전성) → 해결(선명함) → 해결(보존성) → 증명(화질 차별화) → 감성 연결(디자인 가치) → 비교(가격 대비 우위) → 공감(타겟 재확인) → 증명(서비스 신뢰) → 증명(배송 신뢰) → CTA(행동 유도)**
>
> **감정 여정:** 고객의 3가지 불안(가격/화질/안전) 공감 → 브랜드 약속 제시 → 구체적 해결책 3단계 → 차별화 증명 → 감성적 가치 연결 → 신뢰 구축 → 최종 구매 유도
>
> 경쟁사와 차별화된 전략: **'3가지 페인포인트 순차 공감'**으로 시작해 고객 감정에 깊이 공명한 후, '브랜드 약속(후킹)'을 중심축으로 삼아 3가지 해결책을 체계적으로 제시합니다. 특히 경쟁사의 최대 약점인 '화질 저하'를 역이용한 증명 구간을 배치하여 신뢰도를 극대화했습니다.

| 단계 | 반영 여부 | 반영 예시 |
|------|------------|-----------|
| 1단계(페르소나) | ✅ | "아이가 만지다 떨어뜨려" (박서윤의 안전 니즈), "세련된" "선명한" (브랜드 톤 3가지 모두 반영) |
| 2단계(상품 특징) | ✅ | FACT: "충격 저항성", "UV 차단", "뮤지엄급 고화질", "프레임리스" / EMOTION: "안전한 일상", "세련된 공간" |
| 3단계(리뷰 인사이트) | ✅ | "스튜디오 30만 원", "뿌옇게", "노이즈", "꼼꼼한 보정", "빠른 배송" 등 고객 실제 언어 다수 인용 |
| 4단계(SEO 키워드) | ✅ | "아크릴액자", "가족사진액자", "웨딩액자", "주문제작" 자연스럽게 구간 1, 10, 11, 14에 배치 |
| 5단계(경쟁사 분석) | ✅ | 피앙세포토의 '감성→정보' 구조 참조, '화질 약점' 역이용 전략, 인테리어 감성 톤 활용 |
| 6단계(소구점) | ✅ | USP 3가지(Safe/Vivid/Refined) 구간 4에서 통합 제시 후, 구간 5~9에서 각각 상세 전개 |

## 4️⃣ 브랜드 일관성 체크

| 항목 | 평가 |
|------|------|
| SEO 키워드 자연스럽게 포함 | ✅ (구간 1, 10, 11, 14에 핵심 검색어 유기적 배치, 과도한 키워드 나열 지양) |
| 브랜드 톤(세련·절제·자신감) 유지 | ✅ (과장 없는 팩트 기반 표현, "안전한", "선명한", "세련된" 일관성 있게 반복) |
| 감성/정보 문장 비율 | 감성 **8** : 정보 **6** (페인포인트 공감에 집중하되, 해결책은 명확한 정보로 뒷받침) |
| CTA 명확성 | ✅ (구간 14 "주문제작" + "특별하게 완성하세요" 직접적 행동 유도 + SEO 키워드 결합) |

---

**[전략적 차별화 핵심]**

**1. 3단계 공감 → 1단계 약속 구조**
경쟁사가 즉시 증명(리뷰 수, 정품 인증)으로 전환하는 것과 달리, MAKE IT는 고객의 세 가지 핵심 불안(가격/화질/안전)에 충분히 공감한 후, 브랜드 약속(구간 4)을 중심축으로 배치하여 신뢰의 출발점을 만들었습니다.

**2. 경쟁사 약점의 전략적 활용**
3단계 리뷰 분석에서 도출된 '화질 저하'라는 경쟁사 최대 약점을 구간 8에서 직접 언급하며, "MAKE IT는 다릅니다"라는 명확한 차별화 메시지로 전환했습니다.

**3. 데이터 기반의 고객 언어 활용**
"30만 원", "뿌옇게", "노이즈", "꼼꼼한 보정" 등 실제 리뷰에서 추출한 고객 언어를 그대로 사용하여 공감의 진정성을 극대화했습니다.

**4. SEO 키워드의 자연스러운 통합**
"안전한 아크릴액자 주문제작, 가족사진" 등 핵심 검색어를 CTA 구간에 유기적으로 결합하여 검색 최적화와 브랜드 메시지를 동시에 달성했습니다.
